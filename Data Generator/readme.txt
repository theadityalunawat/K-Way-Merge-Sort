How to run ?
java -jar Generator.jar <noofrecords> <sizeofcol1> <sizeofcol2> .... <sizeofcol3>

Output:
Two files will be generated metadata.txt and the input.txt
metadata.txt will contains:
col1,sizeofcol1
col2,sizeofcol2
...
...
coln,sizeofcoln

input.txt will contains:
column data separated by a single space.
